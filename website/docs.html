Quick Integration Example

JavaScript Fetch

const response = await fetch(
  'https://api.formbridge.dev/submit',
  {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'X-Api-Key': 'your_key'
    },
    body: JSON.stringify({
      name: 'John',
      email: 'john@example.com',
      message: 'Hello!'
    })
  }
);

const data = await response.json();
console.log('ID:', data.id);

Common Questions

How much does FormBridge cost? ‚ñ∂

FormBridge is free on the AWS free tier for typical usage. Lambda, DynamoDB, and SES all have generous free allocations that cover most small-to-medium projects.

Can I use FormBridge in production? ‚ñ∂

Yes! FormBridge is built with production in mind. It includes rate limiting, error handling, monitoring via CloudWatch, and is backed by AWS's 99.95% SLA.

Do I need to expose my API key on the frontend? ‚ñ∂

For development, yes. For production, it's recommended to use a backend proxy or create form-specific keys with limited permissions. See the HMAC signing docs for additional security.

Is my data encrypted in transit and at rest? ‚ñ∂

Yes. Submissions use HTTPS, and DynamoDB data is encrypted at rest by default with AWS-managed keys.

How do I export my submissions? ‚ñ∂

Use the /export endpoint to download a CSV file of all submissions. You can also view real-time stats in the analytics dashboard.
